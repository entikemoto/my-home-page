## 検索方法

### Claude Code で検索

```
.claude/docs/research/ にある {キーワード} に関するリサーチ結果を探して
```

### コマンドで検索

```bash
# ファイル名で検索
ls .claude/docs/research/ | grep -i {キーワード}

# 内容で検索
grep -r "{キーワード}" .claude/docs/research/
```

## リサーチ結果の追加ルール

### 1. ファイル名の命名規則

```
{トピック}-{サブトピック}.md
```

例：
- `fastapi-authentication.md`
- `react-state-management.md`
- `python-testing-frameworks.md`

### 2. ファイルのフォーマット

```markdown
# {トピック} リサーチ結果

**調査日**: YYYY-MM-DD
**調査ツール**: Gemini CLI
**調査者**: {名前 or AI}

## 概要

1-2段落の要約

## 主要ポイント

-

## 比較（複数の選択肢がある場合）

| 項目 | A | B | C |
|------|---|---|---|
| | | | |

## 推奨

このプロジェクトでは {X} を推奨。理由：

## コード例

```{language}
```

## 参考リンク

-

## 注意点

-
```

### 3. INDEX.md への追記

新しいリサーチを追加したら、この INDEX.md の一覧に追記してください。
