## コスト削減のルール

### Codex（有料）を使うべき場面

- [ ] 設計判断（複数の選択肢を深く分析）
- [ ] 複雑なデバッグ（原因が不明）
- [ ] アーキテクチャレビュー

### Codex を使わなくてよい場面

- [ ] 単純な質問（Claude Code で十分）
- [ ] 同じ質問の繰り返し（結果を保存して再利用）
- [ ] 調べ物（Gemini を使う）

### Gemini（無料）を積極的に使う場面

- [ ] ライブラリ/ドキュメント調査
- [ ] 最新情報の検索
- [ ] PDF/動画の解析
- [ ] コードベース全体の分析

## アラート基準

| 状況 | アクション |
|------|-----------|
| 週間 Codex 利用 > 50回 | 使い方を見直し |
| 月間コスト > 予算の80% | Lead に報告 |
| 月間コスト > 予算 | 残り期間は Codex 利用を制限 |
